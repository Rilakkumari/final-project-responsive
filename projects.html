Weather App

The Weather App is a much more complex project than the Japan Page below. It was my final project of the SheCodes Plus workshop and took quite a few weeks to finish. Connected to an API, it is fully functional with all cities all over the world, showing you the current time, temperature, hourly and daily forecast as well as wind speed and humidity. This project includes HTML, CSS, Javascript and API integration.

Japan Project

The Japan page was my first SheCodes Project, part of the SheCodes Basics workshop. It is a very simple landing page about Japan, which is the country I love and live in. It includes all the HTML and CSS that we learned: paragraphs, lists, links, borders, backgrounds, pictures and more. I also included a video, because I think it makes the page a bit more interesting and shows the true beauty of Japan.
